# Lan Party - Proiect SD 2026

## Descriere

Proiectul simuleaza un turneu .Programul mai intai elimina echipele cu cel mai mic punctaj pana ajung la cea mai apropiata putere a lui 2 .Apoi organizeaza meciurile eliminatorii si genereaza clasamentul final.

## structuri de date folosite:

1.Petru task 1 am folosit o lista in care am introdus toate echipele .Fiecare echipa continea numele echipei ,Numele jucatorilor si cat de buni is.
2.pentru task 2 am folosit cozi pentru a simula meciurile 
3.Tot pentru task 2 am folosit stive pentru a stoca invingatorii si invinsii care ulterior sunt eliberati din memorie.
4.Pentru task 3 am folosit arbori binari de cautare pentru a clasa cele mai bune 8 echipe.

## Dificultati
